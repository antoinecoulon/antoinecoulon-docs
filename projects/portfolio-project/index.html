Install​

pnpm​

Avoir pnpm installé:

pnpm -v

Sinon:

npm install -g pnpm@latest-10

Vite​

pnpm create vite@latest portfolio --template react-ts

Tailwind​

pnpm add tailwindcss @tailwindcss/vite

Remplacer tout dans src/index.css par:

src/index.css
@import "tailwindcss";

TS config​

Ajoutez compilerOptions:

{
  "files": [],
  "references": [
    {
      "path": "./tsconfig.app.json"
    },
    {
      "path": "./tsconfig.node.json"
    }
  ],
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
{
  "compilerOptions": {
    // ...
    "baseUrl": ".",
    "paths": {
      "@/*": [
        "./src/*"
      ]
    }
    // ...
  }
}

Vite config​

pnpm add -D @types/node
vite.config.ts
import path from "path"
import tailwindcss from "@tailwindcss/vite"
import react from "@vitejs/plugin-react"
import { defineConfig } from "vite"

// https://vite.dev/config/
export default defineConfig({
  plugins: [react(), tailwindcss()],
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
})

Shadn​

pnpm dlx shadcn@latest init

Vous pouvez maintenant ajouter des composants:

pnpm dlx shadcn@latest add button

Cette commande ajoute le composant Button au projet:

src/App.tsx
import { Button } from "@/components/ui/button"

function App() {
  return (
    <div className="flex flex-col items-center justify-center min-h-svh">
      <Button>Click me</Button>
    </div>
  )
}

export default App
